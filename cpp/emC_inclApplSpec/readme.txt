This folder contains sub folders for sets of application settings.
One of this sub folder may part of the users include path:
Any of this sub folder contains a 
  <applstdefJc.h> 
for proper usage for the selected settings and some more include files.
The user can copy this files and use it as template for its own approach.

The <applstdefJc.h> may include some files from source/<FwConv_h/file.h>.
